Acessando a API

Passo 1: Obtenha sua chave de API

Para começar, inscreva-se para um teste gratuito e receba sua chave de acesso à API exclusiva. Essa chave é necessária para autenticação e pode ser encontrada no seu e-mail de boas-vindas.
Passo 2: Faça sua primeira chamada de API

Utilize o seguinte endpoint para fazer sua chamada inicial à API e obter placares em tempo real:

GET https://statpal.io/api/API_VERSION/SPORT_NAME/livescores?access_key=YOUR_ACCESS_KEY
Por favor, substitua os seguintes itens:

API VERSIONcom um dos (v1, v2)
SPORT_NAMEcom seu esporte
YOUR_ACCESS_KEYcom sua chave de API real.

Etapa 3: Integre com seu código

Veja como você pode fazer a mesma solicitação usando a requestsbiblioteca do Python:


import requests

response = requests.get(
    "https://statpal.io/api/v2/soccer/matches/live",
    params={"access_key": "YOUR_ACCESS_KEY"}
)

data = response.json()
print(data)
                        
Lembre-se de substituir YOUR_ACCESS_KEYpela chave de acesso que você obteve na Etapa 1.
